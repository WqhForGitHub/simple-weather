### 技术框架
该应用是基于 Vue + vuex 实现的，页面的 UI 则是使用了 Antd-design-vue 库来完成的。

### 特点
* 简洁的界面及展示必要的天气信息
* 支持县级、区级天气
* 自定义查询天气支持中文、拼音输入
* .....

### 天气接口
* 天气接口: 该天气数据 API 是由和风天气提供的，key 直接暴露出来了，由于是免费版本，所以接口的调用次数和频率有一定的限制，建议去官网申请一个。
